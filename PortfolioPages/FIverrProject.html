  • Project date: 14 April, 2023

Client Requirements

  1. Game should be created using C++ and not blueprint.
  2. The game should be completed within 3 to 4 days.
  3. The game should be a first-person shooter (FPS) with a start HUD that includes a start button and controls for the player.
  4. The player character should have a gun and follow a path, such as a maze.
  5. AI-driven enemies should come towards the player and be able to path towards the player without any issues.
  6. The gun should be able to fire and hit the enemies. The enemies should disappear when their health is gone, but their health should be displayed.
  7. A timer should be displayed at the top of the screen to show how fast the player can beat the level.
  8. When the player reaches the end of the level, a "you win" screen should be displayed along with a "try again" button.
  9. Something should indicate that the player is about to reach the end of the level, such as red smoke or a banner.
  10. The player should have health and be able to die if hit enough by the enemies. A "you died" screen should be displayed.
  11. The enemies should be triggered by the player as they move through the set path.
  12. The project name should be "Project_1".
  13. The code should be commented as needed.
  14. The game can be developed using either UE4 or a different engine.
